The machine learning examples in this presentation will utilize the IRIS data set built into R and kaggle’s ‘fruit with colors’ data set. Iris species will be predicted by sepal and petal dimensions, while fruit type will be predicted by weight, dimensions, and color score.

I. k-nearest neighbors

k-nearest neighbors (KNN) is a supervised machine learning algorithm, utilizing a non-parametric classification method based on Euclidean distance and a plurality among neighbors to make a prediction (ties are broken at random). KNN can also be used for regression, where the output is the average of values of k-nearest neighbors.

We’ll normalize the data between 0 and 1 to give each feature equal importance:

normalize <- function(x) {
  return((x - min(x)) / (max(x) - min(x)))
}

Create the IRIS training and testing data:

irisLabels <- iris[,5] # remember the original species labels
set.seed(1234) # for reproducibility
# indexes of random sample representing 75% of original data
irisIndex <- sample(nrow(irisNormal), size = floor(0.75 * nrow(irisNormal)))
trainingData <- irisNormal[irisIndex,1:4] # training data
trainingLabels <- irisNormal[irisIndex,5] # labels for the training data
testingData <- irisNormal[-irisIndex,1:4] # testing data 25% of original data
testingLabels <- irisLabels[-irisIndex] # labels for testing data

Visualize our data:

trainReference <- cbind(trainingData,trainingLabels) # training data frame
# scatterplot matrix for training data, a look at our neighbors
pairs(trainReference[,1:4], col = trainReference[,5], oma=c(3,3,3,15))
par(xpd = TRUE)
legend("bottomright", fill = unique(trainReference[,5]), 
       legend = c(levels(trainReference[,5])))

k-nearest neighbor with IRIS:

# make sure k is odd
k1 <- round(sqrt(nrow(trainingData)))
if ((k1 %% 2) == 0) {k1 <- k1 + 1} 
irisPrediction1 <- knn(train=trainingData, # training data
                      test=testingData, # testing data
                      cl=trainingLabels, # training labels
                      k=k1) # how many neighbors
pander(table(testingLabels,irisPrediction1)) # confusion matrix
  setosa versicolor virginica
setosa 15 0 0
versicolor 0 12 0
virginica 0 1 10

The above rows represent the true classification of the test data and the columns represent the prediction. Note only 1 failed prediction.

What was the accuracy?

cf1 <- confusionMatrix(testingLabels,irisPrediction1)
print(paste(signif(cf1$overall[1],digits=3)*100,"% accurate."))
## [1] "97.4 % accurate."

II. C5.0 classification tree model

C5.0 is a supervised machine learning algorithm, developed by Ross Quinlan, used to generate decision trees. Information entropy is computed as a value which determines optimal splits of purer classes in the data.

III. Recursive partitioning classification tree model

CART is another supervised machine learning algorithm, developed by Leo Breiman, J. H. Friedman, R. A. Olshen, and C. J. Stone, used to generate decision trees. A computed Gini coefficient is computed and minimized to determine optimal splits in the data.

Recursive partitioning with IRIS:

model3 <- rpart(Species ~ ., data=iris[irisIndex,],method="class")
rpart.plot(model3)

Confusion matrix:

irisPrediction3 <- predict(model3, iris[-irisIndex,],type="class")
pander(table(testingLabels,irisPrediction3))
  setosa versicolor virginica
setosa 15 0 0
versicolor 0 11 1
virginica 0 3 8

What was the accuracy?

cf5 <- confusionMatrix(iris[-irisIndex,5],irisPrediction3)
print(paste(signif(cf5$overall[1],digits=3)*100,"% accurate."))
## [1] "89.5 % accurate."

Recursive partitioning with fruit:

model4 <- rpart(fruit_name ~ ., 
                data=fruit[fruitIndex,], method="class")
par(mar = c(5.1, 4.1, 4.1, 2.1))
rpart.plot(model4,legend.x=.75)
